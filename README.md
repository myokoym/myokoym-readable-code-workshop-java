# 実践リーダブルコード

レシピ管理プログラム

## 開発言語

Java

## 使い方

Recipe.javaのmainメソッドを実行してください。

### EclipseなどのIDE

RecipeクラスをJavaアプリケーションとして実行してください。

### コマンドライン

Recipe.javaをjavacコマンドでコンパイルしてからjavaコマンドで実行してください。

```bash
$ javac -d bin/ src/net/myokoym/recipe/*.java
$ java -cp bin/ net.myokoym.recipe.Recipe
```
